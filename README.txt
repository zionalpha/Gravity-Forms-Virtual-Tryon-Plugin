Add virtual try-on capabilities to your GravityForms using Replicate API.

== Description ==

This plugin adds virtual try-on functionality to GravityForms using the Replicate API. Users can upload their face image and a model image to see how they would look in different outfits.

Features:
* Seamless GravityForms integration
* Real-time image processing
* Customizable field mapping
* Easy setup and configuration

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/virtual-tryon-gravityforms` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure the plugin in the Gravity Forms settings

== Frequently Asked Questions ==

= What is required to use this plugin? =

* WordPress 5.8 or higher
* GravityForms 2.5 or higher
* Replicate API token
* PHP 7.4 or higher
